# File Structure

```
ecommerce-store/
│── index.html              # Homepage (featured products)
│── products.html           # All products listing
│── cart.html               # Shopping cart page
│── checkout.html           # Checkout simulation
│── about.html              # About/Contact page

│── css/
│   └── style.css           # Global styles
│   └── cart.css            # Cart-specific styles
│   └── responsive.css      # Media queries for mobile/tablet

│── js/
│   └── products.js         # Product data (array of objects)
│   └── cart.js             # Cart functionality (add/remove/update)
│   └── search.js           # Search & filter logic
│   └── checkout.js         # Checkout page logic
│   └── main.js             # Navigation, animations, utilities

│── images/
│   └── logo.png
│   └── products/           # Product images

│── README.md               # Project documentation

```
